# Machine Learning - Customer Purchase Prediction

## Objective
To build a machine learning model that predicts whether a customer will purchase a product.

## Algorithm Used
Logistic Regression

## Dataset
Dataset contains:
- Age
- Salary
- Purchased (Target)

## Results
- Model trained successfully
- Accuracy: 1.0
- Predictions generated correctly

## Visualization
Scatter plot showing customer distribution based on Age and Salary.

## Conclusion
This project demonstrates how machine learning can predict customer buying behavior using Logistic Regression.
